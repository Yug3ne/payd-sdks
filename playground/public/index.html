Getting Started

Interactive documentation for the payd-node-sdk. Each card below maps to an SDK method — expand it to see the code example, required fields, and try it live.

Installation & Setup 
npm install payd-node-sdk

import { PaydClient } from "payd-node-sdk";

const payd = new PaydClient({
  apiUsername: "your_api_username",
  apiPassword: "your_api_password",
  defaultCallbackUrl: "https://your-server.com/webhook",
  defaultUsername: "your_payd_username",  // optional
  walletType: "local",                   // "local" or "USD"
});

Configuration

API Credentials

configure

Enter your Payd API credentials to initialize the SDK for this browser session. These map to PaydClient constructor options.

SDK Code 
const payd = new PaydClient({
  apiUsername: "your_api_username",
  apiPassword: "your_api_password",
  defaultUsername: "your_payd_username",
  defaultCallbackUrl: "https://your-server.com/webhook",
  walletType: "local",   // "local" (KES) or "USD"
  baseUrl: "https://api.payd.money",  // optional
});

Balances

Get All

idle

Retrieves fiat and onchain wallet balances for a Payd username.

SDK Code & Fields 
const balances = await payd.balances.getAll("my_payd_user");

console.log(balances.fiatBalance);    // { balance: 1000.00, currency: "KES" }
console.log(balances.onchainBalance); // { balance: 500, currency: "USD" }
ParameterTypeRequired
usernamestringOptional — falls back to defaultUsername
Returns: { fiatBalance, onchainBalance }
Networks

Discover

idle

Discovers available mobile money and bank networks for a country. Use results to fill Pan-African collection/payout forms.

SDK Code & Fields 
const networks = await payd.networks.discover("receipt", "+234");

console.log(networks.mobile); // Mobile money networks
console.log(networks.banks);  // Bank networks

const mtn = networks.findMobile("MTN");
console.log(mtn.minAmount, mtn.maxAmount);

// Get params for a payment request
const params = mtn.toPaymentParams();
// { networkCode, channelId, transactionChannel, providerName, providerCode }
ParameterTypeRequired
transactionType"receipt" | "withdrawal"Yes
dialCodestring (e.g. "+234")Yes
Returns: { mobile: Network[], banks: Network[] }

Pan-African

idle

Collects payments across African countries via mobile money or bank. Run Networks Discover first to get valid networkCode and channelId.

SDK Code & Fields 
// Step 1: Discover available networks
const networks = await payd.networks.discover("receipt", "+234");
const opay = networks.findMobile("OPay");

// Step 2: Collect payment
const result = await payd.collections.panAfrican({
  username: "my_payd_user",
  ...opay.toPaymentParams(),
  accountName: "phone",
  amount: 1800,
  phoneNumber: "+2340000000000",
  accountNumber: "+2340000000000",
  narration: "Order #1234",
  currency: "NGN",
  callbackUrl: "https://example.com/webhook",
});
ParameterTypeRequired
usernamestringOptional — uses default
accountName"phone" | "bank"Yes
amountnumberYes
phoneNumberstring (+country code)Yes
accountNumberstringYes
networkCodestring (UUID from discover)Yes
channelIdstring (UUID from discover)Yes
transactionChannel"phone" | "bank"Yes
currencystring (e.g. "NGN")Yes
narrationstringYes
callbackUrlstringOptional — uses default
redirectUrlstringOptional — for South Africa
Returns: { transactionReference, success }

Webhooks
Webhooks

Parse Event

idle

Paste a raw webhook JSON payload to see the normalized event the SDK produces.

SDK Code & Fields 
app.post("/webhook", (req, res) => {
  const event = payd.webhooks.parseEvent(req.body);

  if (event.isSuccess) {
    console.log(`Payment ${event.transactionReference} succeeded!`);
    console.log(`Type: ${event.transactionType}`); // "receipt", "withdrawal"
    console.log(`Amount: ${event.amount}`);
  } else {
    console.log(`Payment failed: ${event.remarks}`);
  }

  res.status(200).send("OK");
});
Returns: { isSuccess, transactionReference, transactionType, amount, remarks, ... }
